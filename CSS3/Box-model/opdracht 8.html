Welke CSS property kun je gebruiken om ervoor te zorgen dat de waarde van width en height de totale breedte en hoogte is van het blok van de voorgaande opdracht?

The width property sets the width of an element. The width of an element does not include padding, borders, or margins! Note: The min-width and max-width properties override the width property.

The height property sets the height of an element. The height of an element does not include padding, borders, or margins!
