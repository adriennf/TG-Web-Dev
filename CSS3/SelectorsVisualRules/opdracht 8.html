Cascade and inheritance

Referenties:

  • MDN moz://a
  • Shay Howe

Onder cascade hebben we over dat de stijlpagina wordt ingeladen van boven naar beneden. Het kan ook overgeschreven worden wanneer verschillende soorten selectors worden gebruikt, dit ligt aan de specificiteitsgewicht. Max specificiteitsgewicht: !important, dan ID, class en type selector. Hoe hoger het specificiteitsgewicht van een selector, hoe meer superioriteit de selector krijgt in een stylingconflict. Een combinatie van selectoors lezen we van rechts naar links.

The inherit keyword geeft aan dat een eigenschap zijn waarde moet erven van het bovenliggende/ parent element. The inherit keyword can be used for any CSS property, and on any HTML element.
