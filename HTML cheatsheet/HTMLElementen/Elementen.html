Image

Voor afbeeldingen voor het web zijn 4 geschikte formaten:

gif: geschikt voor afbeeldingen met vlakken, zoals logo's. Supported: Chrome, Edge, Firefox, IE, Opera, Safari.

jpeg (jpg): geschikt voor afbeeldingen met veel kleuren, zoals foto's. Support: Chrome, Edge, Firefox, IE, Opera, Safari.

png: geschikt voor alle soorten afbeeldingen (combinatie van gif en jpeg). Support: Chrome, Edge, Firefox, IE, Opera, Safari.

svg: geschikt voor schaalbare afbeeldingen, zoals logo's. Support: Chrome, Edge, Firefox, IE, Opera, Safari.

The required alt attribute provides an alternate text for an image, if the user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader). The value of the alt attribute should describe the image. If a browser cannot find an image, it will display the value of the alt attribute.
